<h1>UniPay</h1>
UniPay is a full-stack wallet-based digital payment system designed for campus environments. It enables secure and seamless transactions between students and merchants while introducing automated rewards and discount mechanisms to improve engagement and usability.

<h3>Features</h3>
<ol>
  <li>Secure user authentication and session management</li>
  <li>Wallet-based digital payments</li>
  <li>Automated reward and discount logic</li>
  <li>Transaction history tracking</li>
  <li>Role-based system (students, merchants,faculty)</li>
  <li>Fast and lightweight Flask backend</li>
</ol>
